# sms-spam-classifier

📌 Project Overview
     This project is a Machine Learning-based SMS Spam Classification system that predicts whether a message is Spam or Ham.

⚙️ Technologies Used
     Python
     Scikit-learn
     Pandas
     NumPy
     Nltk
     Wordcloud
     TF-IDF Vectorizer
     Multinomial Naive Bayes

🔍 Workflow
     Data Cleaning & Preprocessing
     Text Vectorization using TF-IDF
     Model Training using Multinomial Naive Bayes
     Model Evaluation

📊 Model Performance
     Achieved 97% accuracy on test data through effective feature extraction and preprocessing.
